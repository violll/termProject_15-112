Instructions:

You must have PIL installed to run this game.
Open and build mainApp.py to open the main screen.

Do not resize the window
Do not modify the highScoreList.txt file

Click to select one of the three buttons
Click a text input field (like the number of waves rectangle on the generate screen) to be able to type in it.
Press escape to return to the main menu from any screen
While in the game mode:
	press 1 to reset the level in the tutorial mode
	press 1 to generate a new random level with the same difficulty and number of waves
	press p to unpause / pause the game -- it initially starts paused so you have time to place towers before enemies appear
	Click on a tower to select it in the sidebar -- select only one tower at a time
	   - its outline will be gold when selected
	   - click a green square on the map to place the tower
        The enemies come down the tan path towards the base -- your goal is to protect the base
